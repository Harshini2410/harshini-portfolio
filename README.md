# Portfolio Website - Nadendla Poornima Harshini

A modern, responsive single-page portfolio website showcasing skills, projects, and certifications with a sleek UI/UX design.

## Features

-  **Modern Design**: Dark theme with professional color scheme (navy, teal, gold accents)
-  **Fully Responsive**: Optimized for all devices (desktop, tablet, mobile)
-  **Smooth Animations**: Fade-in effects, hover interactions, and smooth scrolling
-  **Performance**: Lightweight vanilla JavaScript, no heavy frameworks
-  **Accessible**: Semantic HTML with proper ARIA labels
-  **SEO Ready**: Meta tags and semantic structure

## Sections

1. **Hero/Bio Section** - Introduction with name, role, contact information, and professional summary
2. **About Section** - Detailed background, education, and experience
3. **Skills Section** - Categorized technical skills with visual tags
4. **Projects Section** - Interactive project cards with tech stack and links
5. **Certifications Section** - Professional certifications with issue dates and credential IDs
6. **Footer** - Contact links and social media icons

## Tech Stack

- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter)

## Setup & Deployment

### Local Development

1. Clone or download the repository
2. Open `index.html` in a modern web browser
3. No build process required - it's ready to use!

### File Structure

```
portfolio/
├── index.html      # Main HTML file
├── styles.css      # All styling
├── script.js       # JavaScript functionality
└── README.md       # This file
```

### Deployment Options

#### GitHub Pages
1. Create a new repository on GitHub
2. Upload all files
3. Go to Settings > Pages
4. Select branch (usually `main`) and `/root` folder
5. Your site will be live at `https://yourusername.github.io/repository-name`

#### Netlify
1. Drag and drop the project folder to [Netlify Drop](https://app.netlify.com/drop)
2. Your site is instantly live!

#### Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Run `vercel` in the project directory
3. Follow the prompts

#### Other Options
- Firebase Hosting
- AWS S3 + CloudFront
- Any static hosting service

## Customization

### Colors
Edit CSS variables in `styles.css`:
```css
:root {
    --bg-primary: #0a192f;
    --accent-teal: #64ffda;
    --accent-gold: #ffd700;
    /* ... */
}
```

### Content
Update sections in `index.html`:
- Hero section: Name, title, description
- About: Education, experience details
- Skills: Add/remove skill categories and tags
- Projects: Update project cards with your projects
- Certifications: Add new certifications
